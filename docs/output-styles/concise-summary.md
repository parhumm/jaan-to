## What It Does

Formats skill outputs as scannable summaries. Use this style for quick reviews, executive updates, or when brevity matters more than detail.

---

## Formatting Rules

| Rule | Description |
|------|-------------|
| **Executive summary** | 3-5 bullet points at top |
| **Decisions** | Presented as a table |
| **Action items** | Numbered checklist |
| **No prose** | Tables and bullets only, no paragraphs |

---

## When to Use

- Quick status updates
- Executive briefings
- Meeting preparation summaries
- Any context where readers scan, not read
