# PM Skills

> Product Manager commands for specs and planning.

---

## Available Skills

| Command | Description |
|---------|-------------|
| [/pm-prd-write](prd-write.md) | Generate PRD from initiative |
| [/pm-research-about](research-about.md) | Deep research or add to index |

---

## Role Context

PM skills generate product outputs:
- Product Requirements Documents (PRDs)
- Feature specifications
- Metrics definitions

---

## Common Workflow

1. Start with an initiative idea
2. Run `/pm-prd-write "your idea"`
3. Answer clarifying questions
4. Review and approve PRD
5. Share with team

---

## Output Location

```
jaan-to/outputs/pm/{slug}/prd.md
```
