## What Detect Skills Do

- Audit repo for engineering, design, writing, product, and UX signals
- Produce machine-parseable markdown with YAML frontmatter
- Score findings with SARIF-like evidence blocks and 4-level confidence
- Consolidate into knowledge pack with risk heatmap and unknowns backlog
